# Artefacto Pacientes - Recursos Terapéuticos

## Descripción del Proyecto

Colección completa de recursos terapéuticos y psicológicos diseñados para pacientes y terapeutas. Este proyecto proporciona herramientas estructuradas para el manejo emocional, desarrollo de habilidades y seguimiento del progreso en terapia.

## Estructura del Proyecto

### 📁 Guías (`/guias`)
Contienen orientaciones generales y marco teórico para terapeutas y pacientes.

- **guia_terapeutas.md** - Guía completa para terapeutas sobre el uso de los recursos

### 📁 Ejercicios (`/ejercicios`)
Material práctico para el desarrollo de habilidades emocionales y cognitivas.

- **ejercicio_mindfulness.md** - Técnicas de mindfulness y atención plena
- **diario_reflexion.md** - Plantilla para reflexión personal y registro emocional
- **regulacion_emocional.md** - Estrategias para la regulación emocional
- **restructuracion_cognitiva.md** - Técnicas de reestructuración cognitiva

### 📁 Herramientas (`/herramientas`)
Prácticas y herramientas aplicables en sesión y entre sesiones.

- **tarjeta_coping_rapido.md** - Técnicas de coping para momentos de crisis
- **plantilla_metas.md** - Plantilla para establecimiento de metas terapéuticas
- **plan_seguridad.md** - Plan de seguridad para manejo de crisis
- **segumiento_progreso.md** - Herramienta para seguimiento del progreso

### 📁 Documentos (`/documentos`)
Documentos de referencia y recursos complementarios.

- **relajacion_pacientes.md** - Guía completa de ejercicios de relajación

## Cómo Utilizar estos Recursos

### Para Pacientes
1. **Explora por categorías** según tus necesidades actuales
2. **Utiliza los ejercicios** de forma regular entre sesiones
3. **Registra tu progreso** usando las herramientas de seguimiento
4. **Comparte tus reflexiones** con tu terapeuta en sesión

### Para Terapeutas
1. **Adapta los recursos** según las necesidades individuales
2. **Utiliza como material complementario** en tus sesiones
3. **Personaliza los planes** de seguridad y metas
4. **Monitorea el progreso** mediante las herramientas disponibles
